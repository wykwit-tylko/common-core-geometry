üìö About This Library

Common Core Geometry is a high-performance 3D geometry engine written in Rust and compiled to WebAssembly. It provides a comprehensive set of primitives and operations for computational geometry tasks.

Features:

  • 8 Core Primitives: Point3D, Vector3D, Sphere, Ray, Triangle, Plane, AABB, LineSegment
  • Fast Performance: Native Rust speed in your browser
  • SVG Rendering: Project 3D geometry to 2D with camera support
  • TypeScript Ready: Auto-generated type definitions
  • Small Bundle: Optimized WASM binary (~111KB uncompressed)

üöÄ Quick Start

Install the package from NPM:

npm install @common-core/geometry

Use it in your project:

import init, { Point3D, Vector3D } from '@common-core/geometry';

await init();

const p1 = new Point3D(0, 0, 0);
const p2 = new Point3D(3, 4, 0);
const distance = p1.distanceTo(p2); // 5.0
